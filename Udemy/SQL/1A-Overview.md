# Oveview of Databases

## What is a Database (DB)?
* Databases allow you to `STORE` and `ORGANIZE` data. They are useful for dealing with `LARGE` amounts of data that a conventional spreadsheet may not be able to handle.
* **`SQL`** stands for `Structured Query Language` and can be applied to a variety of DBs or SQL based software.
* SQL is the programming language used to communicate with your DB.
* `PostgreSQL`, `MySQL`, and `MariaDB` are widely used since they are open-source, free, and can be used on multiple platforms.


## Spreadsheets vs. Databases (re: Why use DBs over Spreadsheets)

#### Spreadsheets
* Good for one-time analyses
* Good for quickly charting something out
* Requires a reasonable data set size
* Allows professionals without data analysis skills to work with small sets of data

#### Databases
* Provides data integrity - cannot easily alter the data
* Capable of handling massive, enterprise-scale levels of data
* Capable of quickly combining different data sets
* Allows for automation - can automate steps for re-use
* Capable of supporting data for websites and applications
